# Pomodoro Project

<p>O Pomodoro Project consiste em um site que permite que qualquer usuário utilize o método Pomodoro para aprimorar seus estudos.</p>
<p>A interface buscou ser bem dinâmica e prática para atender àqueles que buscam organizar o seu tempo sem justamente perder tempo com configurações complexas.</p>
<p>O site conta com notificações no navegador e um alarme ao fim de cada tempo.</p>
<p>Para conhecer melhor, [clique aqui](https://kamilamarquess.github.io/Pomodoro_Project/)</p>
